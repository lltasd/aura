- `studio-interior.jpg` - Интерьер студии (белая комната с кушетками)

## 2. Наши кабинеты (6 изображений)

- `cabinet-3.jpg` - Кабинет с большим зеркалом и косметическим столом
- `cabinet-4.jpg` - Кабинет с массажным столом и оборудованием для процедур
- `cabinet-5.jpg` - Светлый косметологический кабинет

## 3. Оборудование (8 изображений)

- `team-photo.jpg` - Групповое фото команды (специалисты в синей медицинской форме)

## Рекомендации:
- Формат: JPG или PNG
- Разрешение: минимум 1200px по большей стороне
- Оптимизация: сожмите изображения для веба (можно использовать TinyPNG или аналогичные сервисы)
- Aspect ratio для кабинетов: 4:3
- Aspect ratio для оборудования: 1:1 (квадрат)

После добавления изображений они автоматически появятся на сайте!
